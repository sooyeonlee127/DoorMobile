# DoorMobile
모바일 청첩장 프로젝트입니다.

### 개발 기간
- 2023년 12월 - 2024년 01월 (2개월)

### 개발 인원
- 1인

### 기능
- 회원의 이미지, 결혼식 정보를 입력하면 자동으로 회원번호 경로로 모바일 청첩장 생성
- 결혼식 안내, 방명록, 갤러리 기능

### 사용기술
- Node.js, React, MongoDB, AWS S3

### 프로젝트 특징
- IntersectionObserver를 이용한 Infinite Scroll
- Session 기반 Authentication & Authorization
